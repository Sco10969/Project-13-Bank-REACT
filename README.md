# Argent Bank - Front React

## Prerequis

- Node.js (version LTS recommandee)
- npm

## Installation et lancement

```bash
# 1) Cloner le repository
git clone https://github.com/Sco10969/Project-13-Bank-REACT.git

# 2) Entrer dans le projet front
cd Project-13-Bank-REACT/Project-13-Bank-REACT

# 3) Installer les dependances
npm i

# 4) Lancer le serveur de developpement
npm run dev
```

L'application front est ensuite disponible sur l'URL affichee par Vite (par defaut `http://localhost:5173`).
